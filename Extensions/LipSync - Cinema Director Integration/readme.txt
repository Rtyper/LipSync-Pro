
///////////////////////////////////
LipSync actions for Cinema Director
///////////////////////////////////


Package Contents:

Actions/LipSyncPause.cs - CinemaActorEvent for pausing a LipSync component.
Actions/LipSyncPlay.cs - CinemaActorEvent for playing a LipSync component.
Actions/LipSyncPlayFromTime.cs - CinemaActorEvent for playing a LipSync component.
Actions/LipSyncResume.cs - CinemaActorEvent for resuming a LipSync component.
Actions/LipSyncStop.cs - CinemaActorEvent for stopping a LipSync component.

*Note*
This package is only compatibile with LipSync Beta 0.5 or higher.
