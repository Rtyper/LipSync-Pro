
////////////////////////////////////////////////////////////
Integration with uSequencer for LipSync Pro
////////////////////////////////////////////////////////////

Package Contents:

Events/LipSyncPlay.cs - USEvent that represents a LipSyncData clip on the timeline. Supports previewing in edit mode.
Events/SetEmotion.cs - USEvent for setting an emotion on a character. 
Events/ResetEmotion.cs - USEvent for resetting the custom emotion on a character. 

*Note*
This package is only compatible with LipSync Pro version 1.2 and above.
