
////////////////////////////////////////////////////////////
Blend System for PolyMorpher
////////////////////////////////////////////////////////////

Package Contents:

PolyMorpherBlendSystem.cs - BlendSystem designed to work with PolyMorpher. Refer to the LipSync documentation on how BlendSystems work.

*Note*
The Blend System is compatible with any standard BlendSystemUser, such as LipSync Pro, LipSync Lite and Eye Controller.
No LipSync Preset is included with this BlendSystem.
