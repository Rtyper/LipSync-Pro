
///////////////////////////////////
LipSync Events for Flux
///////////////////////////////////


Package Contents:

Actions/LipSyncPause.cs - FEvent for pausing a LipSync component.
Actions/LipSyncPlay.cs - FEvent for playing a LipSync component.
Actions/LipSyncPlayFromTime.cs - FEvent for playing a LipSync component.
Actions/LipSyncResume.cs - FEvent for resuming a LipSync component.
Actions/LipSyncStop.cs - FEvent for stopping a LipSync component.
Actions/LipSyncSetEmotion.cs - FEvent for setting the current emotion on a LipSync component.

*Note*
This package is only compatibile with LipSync Beta 0.6 or higher.
