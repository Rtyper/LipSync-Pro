Blend System for Unity Multipurpose Avatar (UMA) 2
Update: 29/07/2016:
	Updated Presets to 1.0 format
////////////////////////////////////////////////////////////

Package Contents:

UMABlendSystem.cs - BlendSystem designed to work with UMA 2 Characters. Refer to the LipSync documentation on how BlendSystems work.
Presets/LipSync Pro Preset.unitypackage - Package containing a LipSyncPreset for UMA 2. Unpack this if using LipSync Pro or Lite.

*Note*
The Blend System is compatible with any standard BlendSystemUser, such as LipSync Pro, LipSync Lite and Eye Controller.
The included preset is only compatible with LipSync Pro and Lite 1.0 or above.
